# POS System Case Study


## Overview
This is a **multi-device POS System case study** designed to provide a seamless user experience across **web and mobile** platforms.  
The project was designed with **intuitive user flows, responsive interfaces, and a clean UI**, and developed using **React Native** (mobile) and **React.js** (web).



## Features
- Multi-device support (mobile & web)  
- Intuitive checkout process  
- User-friendly dashboard for sales and inventory  
- Clean and modern UI design  
- Responsive layouts across devices  



## My Role
As the **Product Designer**, I was responsible for:  
- Designing **user flows and wireframes**  
- Creating **high-fidelity screens** in UIXFig Studio  
- Ensuring **cross-platform consistency**  
- Collaborating with teammates on design decisions  



## Tech Stack
- **Front-end (Web):** React.js  
- **Front-end (Mobile):** React Native  
- **Design:** UIXFig Studio
